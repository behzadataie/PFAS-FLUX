# MB synthetic case archive

This archive supports the technical note manuscript package. It is deliberately synthetic. All coordinates are local synthetic coordinates in metres. No real coordinates, site names, road names, bore identifiers, client names or geographic linkages are included.

Data basis labels used in the CSV files:
- public anchor, and modified: the value is the same order of magnitude as a publicly available Australian PFAS report but has been altered for the synthetic benchmark.
- literature range: the value lies within ranges reported in peer-reviewed or guidance literature.
- synthetic assumption: the value was created to make the benchmark internally consistent and computationally usable.
- guidance/literature derived: the value is a reporting convention or method element drawn from guidance and literature.

The archive is for method demonstration only. It must not be used as a compliance dataset, a site assessment, a risk conclusion, or a representation of any real property.
