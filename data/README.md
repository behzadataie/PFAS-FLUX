# Data folder

`synthetic_case_archive` contains the required CSV inputs for the PFAS-FLUX demonstration. All coordinates are local coordinates in metres. The files are a realistic synthetic benchmark for reproducing the technical note calculations.

Each important input has a paired `*_basis` column wherever practical. Basis labels distinguish public-range anchors, literature ranges, synthetic assumptions, and derived outputs. Replace these CSVs with a project-specific archive using the same column names to apply the method to another case.
